Architecture Deep-Dive

A client-side only (CSO) application interacting directly with the Google ecosystem.

1. High-Level Overview

Cisco IOS AI operates without a custom backend. All intelligence is processed via the Gemini API, and all storage is handled locally or via the user's private Google Drive.

2. Technology Stack

React 19 Vite 6 Tailwind CSS Google Gemini API Google Drive API v3 WebWorkers

3. Data Flow

1. Input -> User Prompt + Media
2. Service -> geminiService (Logic Engine)
3. API -> Google Generative AI (Grounding)
4. Result -> Structured JSON Response
5. Storage -> LocalStorage + Drive Async Patch

4. Determinism Protocols

To ensure configuration accuracy, we enforce low temperature (0.1) and fixed seeds (42) on all documentation-heavy requests.
